# GitHub Issue Sound

Extensão para Chrome que toca um som toda vez que você fechar uma issue no GitHub.

## Instalação

1. Baixe ou clone esta pasta
2. Abra o Chrome e acesse `chrome://extensions`
3. Ative o **Modo do desenvolvedor** (canto superior direito)
4. Clique em **Carregar sem compactação**
5. Selecione a pasta do projeto

Pronto. O som vai tocar automaticamente sempre que você fechar uma issue.

## Como funciona

- Fecha uma issue individualmente (botão "Close issue")
- Fecha pela lista de issues (Mark as > Completed / Not planned)

Ambos disparam o som.

## Arquivos de som

Os arquivos `drumroll.mp3` e `success.mp3` precisam estar na pasta da extensão. Se quiser trocar os sons, substitua os arquivos mantendo os mesmos nomes.
